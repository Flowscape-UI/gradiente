# Security Policy

If you believe you have found a security issue in Gradiente, please report it by opening a GitHub Issue in this repository.

Repository issues:
https://github.com/Flowscape-UI/gradiente/issues

---

## Reporting Guidelines

Please include:

1. Clear description of the issue
2. Steps to reproduce
3. Potential impact
4. Suggested remediation (optional)

---

## Scope

Gradiente is a lightweight gradient processing library and does not currently operate external infrastructure or hosted services.

Security reports should be relevant to:

* Malicious input causing unexpected code execution
* Prototype pollution / unsafe object handling
* Denial of service via malformed input
* Supply-chain / dependency vulnerabilities affecting published package behavior
* Other vulnerabilities materially affecting consumers of the library

---

## Disclosure

Security issues reported through public GitHub Issues may be visible to others immediately.

If you prefer coordinated/private disclosure, please contact the maintainer directly instead.
